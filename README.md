# Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full Stack Developer. Built with pure HTML5, CSS3, and vanilla JavaScript, featuring smooth animations, dark/light theme toggle, and interactive components [file:11][file:12][file:13].

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices and screen sizes [file:13]
- **Dark/Light Theme**: Toggle between dark and light modes with persistent theme preference using localStorage [file:11]
- **Dynamic Typing Effect**: Animated text that cycles through multiple roles (Full Stack Developer, UI/UX Designer, Problem Solver, Tech Enthusiast) [file:11]
- **Interactive Navigation**: Smooth scrolling navigation with active link highlighting based on scroll position [file:11]
- **Project Filtering**: Filter projects by category (All, Web, Mobile, Design) with smooth animations [file:11]

- **Contact Form**: Functional contact form with EmailJS integration [file:11]
